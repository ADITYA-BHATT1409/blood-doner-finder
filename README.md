# BloodConnect - Emergency Blood Donor Finder

BloodConnect is a modern, responsive web application designed to connect blood donors with those in emergency and immediate need of blood.

## Features
- **Emergency Dashboard:** Live feed of critical blood requirements in your area.
- **Instant Filtering:** Search for donors by specific blood groups and locations instantly.
- **Automated Emergency Requests:** Simple, one-click interface to post an emergency requirement directly to the community platform.

## Project Structure
- `frontend/`: Contains all client-side code (`index.html`, `style.css`, `app.js`). Built entirely natively with zero heavy dependencies for lightning-fast loading during emergencies.
- `backend/`: Reserved for future backend logic and API handling.
- `start.bat`: A quick-start executable for Windows that automatically opens the frontend application in your default browser.
- `requirements.txt`: Python package requirements for future API implementation.

## How to Run
1. Double click the `start.bat` file in your File Explorer.
2. Alternatively, simply open `frontend/index.html` in any web browser (Chrome, Edge, Safari, Firefox).
